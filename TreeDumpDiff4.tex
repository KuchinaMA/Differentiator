\chapter{Нахождение производной на примере некоторой функции}\section{Введение}
Производные функций, оказывается, придуманы не только для того, чтобы удивлять студентов математических факультетов! Они также замечательно подходят для того, чтобы вычислить скорость вашего друга, который завтракает за 5 минут до начала лекции, а также чтобы предсказать, когда ваш утренний кофе остынет до состояния "только для отважных". Так что, поверьте, производные действительно применимы в повседневной жизни, и на вопрос ваших друзей-гуманитариев "А где тебе это в жизни пригодится?" можно найти тысячу и один ответ и даже больше.

Возможно, вы всё ещё думаете, что брать производные - слишком сложная задача? Но выдающиеся люди говорили: "дифференцировать можно и обезьяну научить" (\sout{именно поэтому автор попытался научить компьютер})

Давайте же покажем, что вы лучше обезьян и найдём производную этой тривиальной функции!

\[ \frac{ tan ( 7 \cdot x ) + 78 ^ { x ^ { 16}}}{ ln ( 15 + x )}\]
\section{Вычисление, собственно, производной}
Итак, продифференцируем выражение:
\[ \frac{ tan ( 7 \cdot x ) + 78 ^ { x ^ { 16}}}{ ln ( 15 + x )}\]
После несложных математических преобразований получим\[( x)' =  1\]
Очевидно, что\[( x ^ { 16})' =  16 \cdot x ^ { 15} \cdot 1\]
Неочевидно, но факт:\[( 78 ^ { x ^ { 16}})' =  ln 78 \cdot 78 ^ { x ^ { 16}} \cdot 16 \cdot x ^ { 15} \cdot 1\]
Несложно заметить, что\[( 7)' =  0\]
Неочевидно, но факт:\[( x)' =  1\]
Неочевидно, но факт:\[( 7 \cdot x)' =  0 \cdot x + 7 \cdot 1\]
Тут могла быть очередная гениальная фраза, но я её не придумала\[( tan ( 7 \cdot x ))' =  \frac{ 1}{ cos ( 7 \cdot x ) \cdot cos ( 7 \cdot x )} \cdot ( 0 \cdot x + 7 \cdot 1 )\]
Тут могла быть очередная гениальная фраза, но я её не придумала\[( tan ( 7 \cdot x ) + 78 ^ { x ^ { 16}})' =  \frac{ 1}{ cos ( 7 \cdot x ) \cdot cos ( 7 \cdot x )} \cdot ( 0 \cdot x + 7 \cdot 1 ) + ln 78 \cdot 78 ^ { x ^ { 16}} \cdot 16 \cdot x ^ { 15} \cdot 1\]
Очевидно, что\[( x)' =  1\]
Неочевидно, но факт:\[( 15)' =  0\]
Несложно заметить, что\[( 15 + x)' =  0 + 1\]
Несложно заметить, что\[( ln ( 15 + x ))' =  \frac{ 1}{ 15 + x} \cdot ( 0 + 1 )\]
После несложных математических преобразований получим\[( \frac{ tan ( 7 \cdot x ) + 78 ^ { x ^ { 16}}}{ ln ( 15 + x )})' =  \frac{ ( \frac{ 1}{ cos ( 7 \cdot x ) \cdot cos ( 7 \cdot x )} \cdot ( 0 \cdot x + 7 \cdot 1 ) + ln 78 \cdot 78 ^ { x ^ { 16}} \cdot 16 \cdot x ^ { 15} \cdot 1 ) \cdot ln ( 15 + x ) - ( tan ( 7 \cdot x ) + 78 ^ { x ^ { 16}} ) \cdot \frac{ 1}{ 15 + x} \cdot ( 0 + 1 )}{ ln ( 15 + x ) \cdot ln ( 15 + x )}\]
\section{Упрощение полученного выражения}
Теперь упростим полученное после дифференцирования выражение: 
\[ \frac{ ( \frac{ 1}{ cos ( 7 \cdot x ) \cdot cos ( 7 \cdot x )} \cdot ( 0 \cdot x + 7 \cdot 1 ) + ln 78 \cdot 78 ^ { x ^ { 16}} \cdot 16 \cdot x ^ { 15} \cdot 1 ) \cdot ln ( 15 + x ) - ( tan ( 7 \cdot x ) + 78 ^ { x ^ { 16}} ) \cdot \frac{ 1}{ 15 + x} \cdot ( 0 + 1 )}{ ln ( 15 + x ) \cdot ln ( 15 + x )}\]

Преобразуем полученное выражение\[ \frac{ ( \frac{ 1}{ cos ( 7 \cdot x ) \cdot cos ( 7 \cdot x )} \cdot ( 0 \cdot x + 7 ) + ln 78 \cdot 78 ^ { x ^ { 16}} \cdot 16 \cdot x ^ { 15} \cdot 1 ) \cdot ln ( 15 + x ) - ( tan ( 7 \cdot x ) + 78 ^ { x ^ { 16}} ) \cdot \frac{ 1}{ 15 + x} \cdot ( 0 + 1 )}{ ln ( 15 + x ) \cdot ln ( 15 + x )}\]

После несложных математических преобразований получим\[ \frac{ ( \frac{ 1}{ cos ( 7 \cdot x ) \cdot cos ( 7 \cdot x )} \cdot ( 0 \cdot x + 7 ) + ln 78 \cdot 78 ^ { x ^ { 16}} \cdot 16 \cdot x ^ { 15} \cdot 1 ) \cdot ln ( 15 + x ) - ( tan ( 7 \cdot x ) + 78 ^ { x ^ { 16}} ) \cdot \frac{ 1}{ 15 + x} \cdot 1}{ ln ( 15 + x ) \cdot ln ( 15 + x )}\]

После несложных математических преобразований получим\[ \frac{ ( \frac{ 1}{ cos ( 7 \cdot x ) \cdot cos ( 7 \cdot x )} \cdot ( 0 + 7 ) + ln 78 \cdot 78 ^ { x ^ { 16}} \cdot 16 \cdot x ^ { 15} \cdot 1 ) \cdot ln ( 15 + x ) - ( tan ( 7 \cdot x ) + 78 ^ { x ^ { 16}} ) \cdot \frac{ 1}{ 15 + x} \cdot 1}{ ln ( 15 + x ) \cdot ln ( 15 + x )}\]

Несложно заметить, что\[ \frac{ ( \frac{ 1}{ cos ( 7 \cdot x ) \cdot cos ( 7 \cdot x )} \cdot 7 + ln 78 \cdot 78 ^ { x ^ { 16}} \cdot 16 \cdot x ^ { 15} \cdot 1 ) \cdot ln ( 15 + x ) - ( tan ( 7 \cdot x ) + 78 ^ { x ^ { 16}} ) \cdot \frac{ 1}{ 15 + x} \cdot 1}{ ln ( 15 + x ) \cdot ln ( 15 + x )}\]

Преобразуем полученное выражение\[ \frac{ ( \frac{ 1}{ cos ( 7 \cdot x ) \cdot cos ( 7 \cdot x )} \cdot 7 + ln 78 \cdot 78 ^ { x ^ { 16}} \cdot 16 \cdot x ^ { 15} ) \cdot ln ( 15 + x ) - ( tan ( 7 \cdot x ) + 78 ^ { x ^ { 16}} ) \cdot \frac{ 1}{ 15 + x} \cdot 1}{ ln ( 15 + x ) \cdot ln ( 15 + x )}\]

Преобразуем полученное выражение\[ \frac{ ( \frac{ 1}{ cos ( 7 \cdot x ) \cdot cos ( 7 \cdot x )} \cdot 7 + ln 78 \cdot 78 ^ { x ^ { 16}} \cdot 16 \cdot x ^ { 15} ) \cdot ln ( 15 + x ) - ( tan ( 7 \cdot x ) + 78 ^ { x ^ { 16}} ) \cdot \frac{ 1}{ 15 + x}}{ ln ( 15 + x ) \cdot ln ( 15 + x )}\]

Поздравляем! мы нашли производную!
